# 🇮🇳 SchemeFinder AI

SchemeFinder AI is an intelligent web application that helps Indian citizens discover government schemes they qualify for. Instead of navigating complex government portals, simply describe your situation in plain language (English, Hindi, or Assamese) and get instant, tailored recommendations powered by the Gemini API.

## ✨ Features

* **Natural Language Search:** Describe your demographic, financial, or employment situation in everyday language—no need to know the specific name of a scheme.
* **Multilingual Support:** Built with accessibility in mind, supporting queries in English, Hindi, and Assamese.
* **AI-Powered Matching:** Utilizes the Gemini API to intelligently analyze your prompt and match you with relevant Central and State Government schemes.
* **Instant & Accessible Results:** Get clear, actionable information on eligibility criteria and application processes instantly.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript 
* **Backend:** Node.js, Express (or your preferred backend)
* **AI Engine:** Google Gemini API
